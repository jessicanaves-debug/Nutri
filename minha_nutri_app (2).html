🌐 Colocar o app online

Para acessar pelo celular de qualquer lugar, siga esses passos simples — é grátis e leva 2 minutinhos!

  1. Acesse app.netlify.com/drop
  2. Arraste o arquivo minha_nutri_app.html para a área indicada
  3. Aguarde alguns segundos — o Netlify gera um link automaticamente!
  4. Abra o link no celular e salve na tela inicial (no Safari: compartilhar → "Adicionar à Tela de Início")

✅ Totalmente grátis · ✅ Sem criar conta · ✅ Link permanente · ✅ Funciona como app no celular

⚠️ Lembre de também configurar sua chave de API na aba "Nutri IA" para o chat funcionar.
